
# Movie Web App - c# .NET CORE 6 

## Dependencies

- .net core 6
- EntityFrameWork 
- Identity 
- Nlog
- PagedList
- Mssql

## Run it on your computer

Clone the project

```bash
  git clone
```
Install required packages

```bash
  Nuget Packages
```

Configre Mssql

```bash
    "ConnectionStrings": {
    "DefaultConnection": "Server=Your Server;Database=Your Db;Trusted_Connection=True;MultipleActiveResultSets=true"
  },
```
